API Documentation

How does this even works?, we're glad you asked because we are going to explain it right here

You can also view the Swagger documentation by clicking Here.

Size

The size can be specified using the format Width x Height:

The Height is optional, you can create a square by not providing it:

The maximum image size is 2000 x 2000.

Format

Generated images can have one of the next three formats: jpg, png or gif.

The default format is jpg.

Text

Text can be placed using a query string, the default text is the image size.

The query string parameter name must be "text".

Color

Background and text color can be set using hex values

Colors can either be written with or withouth a hash '#' at the beginning. Both '#fff' and 'fff' are valid.

You can also set the background color only, if done this way, the image background will be generated with the provided custom color but with the default gray for font.

Alternatively, you can use color names to set colors.
